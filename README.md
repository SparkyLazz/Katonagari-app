# 💰 Katonagari

> Personal finance app built for the Indonesian market — track expenses, manage budgets, and take control of your money.

---

## 📖 Overview

**Katonagari** is a mobile personal finance app designed specifically with Indonesian users in mind. Built with Flutter for a smooth cross-platform experience, Katonagari helps you track daily spending, manage budgets, and build better financial habits — all in one place.

The name *Katonagari* reflects the app's identity: a personal, grounded approach to managing money in everyday Indonesian life.

> ⚠️ This project is currently in active development. Features and UI are subject to change.

---

## ✨ Features

- 📊 **Expense Tracking** — Log and categorize your daily transactions quickly
- 💼 **Budget Management** — Set monthly budgets per category and monitor your limits
- 🏦 **Multi-wallet Support** — Manage multiple accounts (cash, bank, e-wallet) in one app
- 📅 **Transaction History** — View and filter past transactions by date or category
- 🇮🇩 **Indonesian Market Focus** — Designed with local currency (IDR), habits, and financial context in mind

## 🛠️ Setup & Installation

### Prerequisites

Make sure you have the following installed:

- [Flutter SDK](https://docs.flutter.dev/get-started/install) (stable channel)
- [Dart](https://dart.dev/get-dart) (comes with Flutter)
- Android Studio or VS Code with Flutter extension
- An Android/iOS emulator or a physical device

### Steps

```bash
# 1. Clone the repository
git clone https://github.com/SparkyLazz/Katonagari-app.git

# 2. Navigate into the project directory
cd Katonagari-app

# 3. Install dependencies
flutter pub get

# 4. Run the app
flutter run
```

> For a release build:
> ```bash
> flutter build apk --release
> ```
